Cycle Counter Privacy Policy

Last updated: June 19, 2026

Cycle Counter ("this extension") respects your privacy. This policy explains what data the extension handles.

1. Data collected

This extension stores only the following settings on your device:

  • Daily increment (dailyIncrement)
  • Cycle length in days (cycleLength)
  • Start date (startDate)
  • Language preference (locale)

No personally identifiable information (name, email, location, etc.) is collected.

2. Data storage

All data is stored locally on your device using the Chrome chrome.storage.local API. No data is transmitted to external servers.

3. Data sharing

This extension does not share data with third parties. There is no integration with advertising networks, analytics tools, or external APIs.

4. Permissions used

  • storage — Save and load your settings
  • alarms — Refresh the badge hourly and at midnight

5. Data deletion

Uninstalling the extension removes all stored data from your device. The "Reset" button in the popup only changes the start date; other settings are kept.

6. Contact

For questions about this policy, please contact us via the Chrome Web Store listing page.

7. Policy changes

This policy may be updated as needed. Significant changes will be noted on the Chrome Web Store listing.

Cycle Counter プライバシーポリシー

最終更新日: 2026年6月19日

Cycle Counter（以下「本拡張機能」）は、ユーザーのプライバシーを尊重します。

1. 収集するデータ

  • 1日あたりの増加量、周期日数、開始日、言語設定

個人を特定できる情報は収集しません。

2. データの保存場所

すべてのデータは端末内（chrome.storage.local）にのみ保存されます。外部送信は行いません。

3. 使用する権限

  • storage — 設定の保存
  • alarms — バッジの定期更新
